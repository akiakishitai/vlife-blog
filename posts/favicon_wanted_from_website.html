🍟はじめに

ショートカットリンクを作成する際にリンク先サイトの favicon を利用したかった。
どうやって取得するのがいいのか調べたときのメモ。

🍻favicon 取得方法

調べた結果、次の3通り覚えとけばよさそう。

とにかく favicon が欲しい

👉 Google Favicon Snatcher を利用する

大きいサイズの favicon が欲しい

👉 Favicon Grabber を利用する

ICO形式そのままの画像が欲しい

👉 直接 favicon にアクセスする

中でも Favicon Grabber がおすすめ。

Google Favicon Snatcher を使う

Google の API で、指定したドメインの favicon を PNG 画像として表示してくれる。

Youtube の favicon 取得例
https://www.google.com/s2/favicons?domain=youtube.com

Favicon Grabber を使う

指定したドメインの favicon をダウンロード可能にしてくれる OSS のオンラインサービス。
https://favicongrabber.com にアクセスし、Webサイトのドメインを入力すればいい。
favicon の中から大きいサイズの画像を返してくれる。

また API も提供しており、favicon のURL一覧を JSON 形式で返してくれる。

API を使う例
https://favicongrabber.com/api/grab/youtube.com

直接 favicon にアクセスする

https://<ドメイン>/favicon.ico にアクセスすれば favicon が表示されるので、それを保存する。
ICO 形式のまま欲しいときに用いる手法。 

https://youtube.com/favicon.ico

favicon の設定方法は次の2通り:

  1. ルートパス直下に favicon.ico 画像ファイルを配置する

  2. link タグで画像のパスを指定する（link rel="icon" href=<favicon画像パス>）

👉 ほとんどのWebサイトでルートパス直下に置く方法を採用しているため、上記手法で favicon を取得できる。
