=== Substitutions

ブロックや `++pass:[]++` マクロでは以下の特殊文字の置換方法を指定できる。

.置換オプション
[cols="^1s,<4"]
|===
|Name |Desc

|none
|Disables substitutions

|normal
|Performs all substitutions except for callouts

|verbatim
|Replaces special characters and processes callouts

|specialchars, specialcharacters
|Replaces <, >, and & with their corresponding entities

|quotes
|Applies text formatting

|attributes
|Replaces attribute references

|replacements
|Substitutes textual and character reference replacements

|macros
|Processes macros

|post_replacements
|Replaces the line break character (+)

|===

.ブロックでの指定
[example]
====
[subs=verbatim]
------
[source,javascript,subs="verbatim,quotes"] // <1>
----
const hello = *"Hello, World!"* // <2>
console.log(hello)
----
------
<1> 特定の文字と吹き出しの置換およびテキストフォーマットを適用
<2> `quotes` により太字に置換される
====

.Result
[source,javascript,subs="verbatim,quotes"]
----
const hello = *"Hello, World!"*
console.log(hello)
----

=== パススルー・マクロ

https://asciidoctor.org/docs/user-manual/#passthroughs[参考]

.パススルー
`++pass:[]++`:: インラインマクロ。明示的に置換するオプションを指定できる。

`+`:: 特殊文字のエスケープのみを行う。置換はサポートしない。

`++`:: 置換はサポートしない。

パススルーマクロ `pass` について、Asciidocが処理する置換オプションの指定は https://asciidoctor.org/docs/user-manual/#applying-substitutions を参考。

[example]
====
++direct pass:none[_italic_]++
++replace pass:q[_italic_]++
====

direct pass:none[_italic_]
replace pass:q[_italic_]
